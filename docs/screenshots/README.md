# Screenshots

Screenshots weren't generated as part of this milestone (this environment has no
network access to run a live scan). To add them yourself:

1. Run a scan: `python -m src.main https://example.com`
2. Take a screenshot of the terminal output (CLI report).
3. Open `reports/example.com/report.html` in a browser and screenshot it too.
4. Save both images here, e.g. `cli-output.png` and `html-report.png`.
5. Reference them in the main `README.md`:
   ```markdown
   ![CLI Output](docs/screenshots/cli-output.png)
   ![HTML Report](docs/screenshots/html-report.png)
   ```
